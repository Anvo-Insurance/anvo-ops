# Submission Checklist: Commercial Auto

## Purpose

Use this checklist to verify that all required information is gathered before submitting a commercial auto risk to carriers. If any required item is missing, request it from the client before submitting.

Cross-reference `carriers/submission_preferences.md` for carrier-specific requirements that go beyond this baseline.

---

## Required ACORD Forms

- [ ] **ACORD 125** — Commercial Insurance Application (general information section)
- [ ] **ACORD 137** — Commercial Automobile Section
- [ ] **ACORD 127** — Business Auto Section (use whichever form the target carrier requires — no single Anvo standard; check `carriers/submission_preferences.md`)

## Vehicle Schedule

- [ ] **Complete vehicle list** with the following for each unit:
  - Year, make, model
  - VIN
  - GVW (Gross Vehicle Weight)
  - Vehicle type / body type (sedan, pickup, box truck, tractor, trailer, etc.)
  - Garage location (address where vehicle is primarily kept overnight)
  - Radius of operations (local, intermediate, long-haul)
  - Current odometer reading (if available)
  - Cost new / stated value / actual cash value
  - Existing damage (if any)

## Driver Schedule

- [ ] **Complete driver list** with the following for each driver:
  - Full legal name
  - Date of birth
  - Driver's license number and state
  - License class (CDL, standard, etc.)
  - Years of driving experience
  - Date of hire
  - MVR (Motor Vehicle Report) — dated within last <!-- TODO: Edward to confirm — typical carrier requirement is 30-90 days --> days
  - CDL medical card expiration (if applicable)

## Loss History

- [ ] **Currently valued loss runs** — minimum 5 years
  - Must be dated within 60 days of submission
  - From all auto carriers in the loss run period
- [ ] **Large loss detail** — for any claim over $25,000 threshold:
  - Date of loss, description, status (open/closed), total incurred, reserves remaining

## Operations Information

- [ ] **Description of business operations** and how vehicles are used
- [ ] **Radius of operations** — local (<50 mi), intermediate (50-200 mi), long-haul (>200 mi)
- [ ] **Cargo hauled** (if applicable) — type, average value per load, max value per load
- [ ] **DOT number** (if applicable)
- [ ] **MC number** (if applicable)
- [ ] **USDOT safety rating** (if applicable)
- [ ] **Any DOT violations in last 3 years?** Details if yes.

## Coverages to Confirm with Client

- [ ] **Liability limits requested** — CSL or split limits
- [ ] **Physical damage** — on which vehicles? Comprehensive and collision deductibles?
- [ ] **Hired auto** — does the insured rent or hire vehicles?
- [ ] **Non-owned auto** — do employees use personal vehicles for business?
- [ ] **Motor cargo / inland marine** (if hauling goods for others)
- [ ] **Uninsured/underinsured motorist** — required in some states
- [ ] **Medical payments**
- [ ] **Trailer interchange** (if applicable)

- [ ] **Safety program documentation** (if in place — helps with pricing)
- [ ] **Telematics/GPS usage** (note if in use — some carriers offer discounts)
- [ ] **Prior policy declaration page** (current auto policy)

## Pre-Submission Verification

Before submitting, confirm:

1. All ACORD forms are fully completed — no blank required fields.
2. Vehicle count on the schedule matches the ACORD app.
3. Driver count on the schedule matches the ACORD app.
4. Loss runs cover the full period required by the target carrier.
5. Any DOT/MC compliance issues are noted in the cover letter.
6. Carrier-specific supplementals are included (see `carriers/submission_preferences.md`).
