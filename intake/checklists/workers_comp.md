# Submission Checklist: Workers' Compensation

## Purpose

Use this checklist to verify that all required information is gathered before submitting a workers' comp risk to carriers. WC underwriting is heavily payroll- and class-code-driven — incomplete payroll data is the #1 reason for submission delays.

Cross-reference `carriers/submission_preferences.md` for carrier-specific requirements.

---

## Required ACORD Forms

- [ ] **ACORD 125** — Commercial Insurance Application
- [ ] **ACORD 130** — Workers' Compensation Application

## Business Information

- [ ] **Legal entity name** and DBA (if different)
- [ ] **Entity type** — sole proprietor, LLC, corporation, partnership
- [ ] **FEIN / Tax ID**
- [ ] **Years in business**
- [ ] **State(s) of operation** — list all states where employees work
- [ ] **Governing class code** and all applicable NCCI (or state bureau) class codes
- [ ] **Description of operations** — detailed enough to validate class code assignment
- [ ] **Does the insured use subcontractors?** If yes:
  - Do they obtain certificates of insurance from all subs?
  - Dollar amount paid to uninsured subcontractors (this gets included in payroll)

## Payroll Data

This is the most critical section. Incomplete payroll data = incomplete submission.

- [ ] **Estimated annual payroll** broken down by:
  - Class code
  - State
  - Full-time vs. part-time headcount per class
- [ ] **Prior year actual payroll** (for audit comparison)
- [ ] **Officers/owners included or excluded?**
  - Names, titles, ownership percentages
  - Elected salary for each (if included) — note state minimums/maximums apply
  - Exclusion forms filed with the state (if excluded)
- [ ] **Seasonal fluctuations?** Peak payroll months and off-season months.
- [ ] **Any employees working in multiple states?** How is payroll allocated?

## Experience Modification

- [ ] **Current experience modification rate (EMR)**
- [ ] **NCCI experience mod worksheet** (or state bureau equivalent)
  - If new business with no mod: note "no mod — new venture" and provide years of management experience
- [ ] **Interstate experience mod** (if operates in multiple NCCI states)

**Note:** Anvo does not currently write WC in monopolistic state fund states (OH, WA, WY, ND). If this changes, add state-specific handling here.

## Loss History

- [ ] **Currently valued loss runs** — minimum 5 years
  - Must include:
    - Date of loss
    - Description
    - Status (open/closed)
    - Total incurred (paid + reserves)
    - Medical vs. indemnity breakdown (if available)
  - Dated within 60 days of submission
- [ ] **Large loss detail** — for any claim over $25,000 threshold:
  - Root cause
  - What corrective action was taken
  - Current status and prognosis

## Safety and Risk Management

These aren't always required but significantly improve pricing:

- [ ] **Written safety program in place?** (If yes, include a copy or summary)
- [ ] **Designated safety officer?** Name and qualifications.
- [ ] **OSHA citations in last 3 years?** Details if yes.
- [ ] **Return-to-work / light duty program?**
- [ ] **Drug testing policy?** Pre-employment, random, post-accident?
- [ ] **Employee training documentation** (for high-hazard classes)

## Pre-Submission Verification

Before submitting, confirm:

1. All ACORD forms are fully completed.
2. Payroll is broken down by class code AND state — no lump sums.
3. Officer/owner inclusion/exclusion is clearly documented.
4. Experience mod is current year and from the correct bureau.
5. Loss runs cover at least 5 years (or life of business if shorter).
6. Subcontractor exposure is documented — insured vs. uninsured subs.
7. The risk falls within the target carrier's WC appetite (see `carriers/carrier_matrix.md`) — check class code eligibility, state restrictions, and mod limits.
