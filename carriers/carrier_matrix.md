# Carrier Matrix

## How to Use This File

This file is the primary reference for carrier appetite and capabilities. When selecting markets for a new submission, read this file to identify which carriers are likely to quote a given risk.

**Decision process:**
1. Identify the line of business needed.
2. Filter carriers that write that line.
3. Check geographic and risk characteristic restrictions.
4. Check premium range fit.
5. Rank by appetite match and submission method efficiency.
6. Cross-reference `carriers/submission_preferences.md` for formatting requirements before submitting.

**Maintenance:** Update this file whenever carrier appetite changes — after underwriter calls, appetite guide updates, or lost quotes that reveal new restrictions. See `carriers/market_notes/` for the logging convention.
