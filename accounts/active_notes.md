# Active Account Notes

## How to Use This File

This file tracks key details and running notes for active Anvo Insurance accounts. When working on any account — renewal, service request, claim, or submission — check this file first for context.

**Maintenance rules:**
- Update this file whenever account details change (carrier switch, new line added, key contact change).
- Add running notes dated `YYYY-MM-DD` for anything that doesn't belong in EZLynx but provides useful context for future interactions (e.g., "client mentioned they're opening a second location in Q3").
- Keep entries concise. This is a quick-reference file, not a full account history — EZLynx is the system of record.
