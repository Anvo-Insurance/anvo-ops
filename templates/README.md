# Templates

## What This Folder Is

This folder holds reusable templates for client-facing and carrier-facing communications: emails, cover letters, proposal formats, and other standard documents used in Anvo's operations.

## File Naming Convention

```
purpose_description.md
```

Examples:
- `client_welcome_email.md`
- `submission_cover_letter.md`
- `renewal_proposal_template.md`
- `bor_letter_template.md`
- `certificate_request_follow_up.md`

## How to Use Templates

1. Identify the appropriate template for your task.
2. Copy the template content.
3. Replace all `{{variable}}` placeholders with actual values.
4. Review the completed message for accuracy and tone before sending.
5. If a template doesn't fit the situation well, adapt it — but preserve the core structure and required information.

## Rules

1. **One template per file.**
2. **Use `{{variable}}` syntax** for all dynamic content.
3. **Include a "Use Case" section** at the top of each template explaining when to use it.
4. **Include a "Variables" section** listing every placeholder and what it should be replaced with.
5. **Keep templates current.** If you notice a template references outdated information or processes, update it.
